# Git Quick Reference for Adding & Deleting Files/Folders

## Adding a File/Folder to GitHub

1. Open VS Code terminal (make sure you're at the repo root).
2. Add the file/folder:
   git add <file_or_folder_name>

   Example:
   git add Intern/ToDoList/

3. Commit the change:
   git commit -m "Added ToDoList project"

4. Push to GitHub:
   git push origin main


## Deleting a File/Folder from GitHub (but keeping it locally)

1. Open VS Code terminal (at the repo root).
2. Remove it from Git tracking:
   git rm -r --cached <file_or_folder_name>

   Example:
   git rm -r --cached self/HTML

3. Add it to .gitignore so it stays untracked:
   echo "<file_or_folder_name>/" >> .gitignore

   Example:
   echo "self/HTML/" >> .gitignore

4. Commit the change:
   git add .gitignore
   git commit -m "Removed self/HTML from repo"

5. Push to GitHub:
   git push origin main
