Life Cycle of Applet 1. init() → Applet loaded and initialized.
2. start() → Applet begins running.
3. paint() → GUI is rendered.
4. stop() → Applet paused (user navigates away).
5. destroy() → Applet removed from memory (browser closes or navigates away permanently).
